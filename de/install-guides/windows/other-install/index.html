Andere downgrading Schritte

Epic

  1. Starte FTL und gehe sicher, dass die Version 1.6.12 ist
  2. Nutze den downgrader für Epic 1.6.12 hier instead of the same files provided in FTL-Hyperspace.zip
  3. Du musst trotzdem FTL-Hyperspace.zip herunterladen wie in der Steam Anleitung, da du die DLLs von dort brauchst!
  4. Folge der Steam Anleitung, aber nutze downgrade.bat und den patch Ordner von der oberen ZIP Datei, anstatt der anderen.
