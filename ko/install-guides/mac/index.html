설치

하이퍼스페이스는 MacOS 를 공식적으로 지원하고 있지 않습니다. 하지만 Dino 의 Hyperspace Wrapper 를 사용하면 Mac 에서 Windows 용 하이퍼스페이스를 사용하실 수 있을 것입니다. Wrapper 의 GitHub 저장소에 있는 README 를 참고하시고, 더 자세한 내용은 아래 영상 가이드를 참조하세요. 바닐라 하이퍼스페이스와 멀티버스를 설치하는 내용이 모두 포함되어 있습니다.

지원이 필요하다면 멀티버스 디스코드 에서 #mac-support-please-bug-dino 채널로 가서 요청하시기 바랍니다 (주: 영문입니다).
